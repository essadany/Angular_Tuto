## 🔥 **Daily Breakdown & Resources**  

### **Day 1: Getting Started & Components**  
📂 `01-getting-started`  
- 📖 Read: [Angular Getting Started](https://angular.io/start)  
- 🔧 Install Angular CLI & Create a new project:  
  ```bash
  npm install -g @angular/cli
  ng new my-angular-learning
  cd my-angular-learning
  ng serve
  ```
- ✅ **Mini-project:** Build a simple UI with **3 components** (Header, Footer, Content).  

📂 `02-components-basics`  
- 📖 Read: [Component Basics](https://angular.io/guide/component-overview)  
- 🔧 Learn: Data Binding (`{{ }}`), Event Binding (`(click)`), Property Binding `[ ]`.  
- ✅ **Mini-project:** **Counter App** (Increment/Decrement buttons).  

---

### **Day 2: Directives & Pipes**  
📂 `03-directives-pipes`  
- 📖 Read: [Directives](https://angular.io/guide/directives), [Pipes](https://angular.io/guide/pipes)  
- 🔧 Learn: `*ngIf`, `*ngFor`, Attribute Directives, Built-in Pipes.  
- ✅ **Mini-project:** **Product List App** (Display a list of products & format price).  

---

### **Day 3: Services & Dependency Injection**  
📂 `04-services-dependency-injection`  
- 📖 Read: [Dependency Injection](https://angular.io/guide/dependency-injection)  
- 🔧 Learn: `@Injectable()`, Creating & Using Services.  
- ✅ **Mini-project:** **Blog App (Mock Data)** – Display blog posts using a service.  

---

### **Day 4: Routing & Navigation**  
📂 `05-routing-navigation`  
- 📖 Read: [Angular Router](https://angular.io/guide/router)  
- 🔧 Learn: RouterModule, Route Parameters, Router Links.  
- ✅ **Mini-project:** **Multi-Page Blog App** – Implement navigation between pages.  

---

### **Day 5: Forms & Validation**  
📂 `06-forms-validation`  
- 📖 Read: [Angular Forms](https://angular.io/guide/forms-overview)  
- 🔧 Learn: Template-driven vs Reactive Forms, Form Validation.  
- ✅ **Mini-project:** **User Registration Form** (with validation).  

---

### **Day 6: API Calls with HTTPClient**  
📂 `07-httpclient-api`  
- 📖 Read: [HTTPClient](https://angular.io/guide/http)  
- 🔧 Learn: Making API Requests, Handling Errors, Observables.  
- ✅ **Mini-project:** **Fetch GitHub Users** – Fetch data from a public API.  

---

### **Day 7: Angular Material (Optional UI Framework)**  
📂 `08-angular-material`  
- 📖 Read: [Angular Material](https://material.angular.io/)  
- 🔧 Learn: Install & Use Material Components (`mat-card`, `mat-button`).  
- ✅ **Mini-project:** **Styled Login Form** using Angular Material.  

---

### **Day 8: Advanced Forms & File Uploads**  
📂 `09-advanced-forms`  
- 📖 Read: [Reactive Forms](https://angular.io/guide/reactive-forms)  
- 🔧 Learn: FormArray, FormGroup, File Uploads.  
- ✅ **Mini-project:** **Multi-step Form**.  

---

### **Day 9: State Management (NgRx or Services)**  
📂 `10-state-management`  
- 📖 Read: [NgRx](https://ngrx.io/) or Service-based state management  
- 🔧 Learn: Global state management techniques.  
- ✅ **Mini-project:** **E-Commerce Cart** – Manage cart state.  

---

### **Day 10: Authentication & Route Guards**  
📂 `11-authentication-guards`  
- 📖 Read: [Route Guards](https://angular.io/guide/router#route-guards)  
- 🔧 Learn: Auth Guard (`canActivate`), JWT Authentication.  
- ✅ **Mini-project:** **Auth System** – Protect routes.  

---

### **Day 11: Performance Optimization**  
📂 `12-optimization`  
- 📖 Read: [Lazy Loading](https://angular.io/guide/lazy-loading-ngmodules)  
- 🔧 Learn: Lazy Loading Modules, Change Detection (`OnPush`).  
- ✅ **Mini-project:** Optimize previous projects.  

---

### **Day 12: Unit Testing**  
📂 `13-testing`  
- 📖 Read: [Testing in Angular](https://angular.io/guide/testing)  
- 🔧 Learn: Jasmine, Karma, Unit Testing Components & Services.  
- ✅ **Mini-project:** **Unit Test Blog App Service**.  

---

### **Day 13-14: Final Project (Capstone)**  
📂 `14-final-project`  
- **Choose a real-world project idea**:  
  - 📌 **Task Manager** (CRUD tasks, auth, API calls).  
  - 📌 **Expense Tracker** (State Management, Auth, API).  
  - 📌 **E-Commerce App** (Product listing, cart, checkout).  
- ✅ **Build the full project!**  

---

### **Day 15: Deployment & Review**  
📂 `15-deployment`  
- 📖 Read: [Angular Deployment](https://angular.io/guide/deployment)  
- 🔧 Learn: Deploying on Firebase, Netlify, Vercel.  
- ✅ **Final Task:** **Deploy your Capstone Project!** 🚀
